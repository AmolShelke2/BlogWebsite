HTML5

The HTML is used to create the skeleton of a web page like where the header goes, where the Images goes Etc.
every thing on a website is a html document. Using HTML you can bind the structure of a website.

-----------------------------------------------------------------------------

CSS

CSS is the language for describing the presentation of Web pages, including colors, layout, and fonts. It allows one to adapt the
presentation to different types of devices, such as large screens, small screens, or printers. CSS is independent of HTML and can be
used with any XML-based markup language.

-----------------------------------------------------------------------------

So in that case you need to learn Javascript, the Javascript is a Programming language basically javascript is used is everywhere
not only in web Development. if you learn javascript then you can make your website, mobile Apps, many more.

-----------------------------------------------------------------------------

Git

what is Git?
Git is a DevOps tool used for source code management. It is a free and open-source version control system
used to handle small to very large projects efficiently. Git is used to tracking changes in the source
code, enabling multiple developers to work together on non-linear development.

-----------------------------------------------------------------------------

Github

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects
from anywhere. This tutorial teaches you GitHub essentials like repositories, branches, commits, and pull requests. ... Create and
use a repository. Start and manage a new branch.
